## Tech Stack
### Website (images below in case website is down or expired)
```
Website: https://commerce-app-ent.onrender.com/
```
### Frontend
```
Language: Typescript
Library + Framework: Vite
Styling: HTML + CSS + Tailwind + DaisyUI
Router: React Router v7
Context: Tanstack Query
Notifications: Toastify
Linter: ESLint + Prettier
Deployed On: Render
```

### Backend
```
Language: Ruby, Ruby on Rails
Database: PostgreSQL
Deployed On: Self-Deployed w/ Docker + Ngrok
```
